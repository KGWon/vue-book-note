# 読書記録アプリ 📚

Vue.js を使用して作成したシンプルな読書記録用 Web アプリです。  
本のタイトル・感想・評価を記録し、一覧で管理できます。  

Composition API を採用し、シンプルでメンテナンスしやすいコンポーネント設計を意識して作成しました。  
ポートフォリオ目的の個人プロジェクトです。

## 🔧 主な機能

- 📖 本の登録（タイトル・感想・評価）  
- 📋 登録した本の一覧表示  
- 💾 ローカルストレージによるデータ保存
